# Core Contracts

Generated by `scripts/check_core_contracts.sh` from contract comments in selected `fasm/core/*.inc` files.

Each public routine in the checked files must have adjacent `; contract:`, `; returns:`, and `; clobbers:` comments.

## `fasm/core/file.inc`

### `file_read_all_fixed`

- contract: rdi = path, rsi = destination buffer, rdx = buffer capacity, rcx = size storage*
- returns: rax = FILE_OK / FILE_ERR_*
- clobbers: rax, rcx, rdx, rsi, rdi, r11

### `file_open_read`

- contract: rdi = path
- returns: rax = fd or -1
- clobbers: rax, rcx, r11

### `file_read_chunk`

- contract: rdi = fd, rsi = buffer, rdx = capacity
- returns: rax = bytes read, 0 EOF, -1 error
- clobbers: rax, rcx, r11

### `file_close`

- contract: rdi = fd
- returns: none
- clobbers: rax, rcx, r11
